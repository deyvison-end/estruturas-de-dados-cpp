# Estruturas de Dados em C++

Projeto com implementação de estruturas fundamentais em C++.

## 📌 Estruturas implementadas
- Tabela Hash simples
- Tabela Hash com encadeamento (tratamento de colisão)
- Lista encadeada

## 🧠 Conceitos aplicados
- Templates em C++
- Funções de hash
- Manipulação de memória
- Estruturas dinâmicas

## ▶️ Como executar
Compile o projeto com:

g++ main.cpp -o programa
./programa
